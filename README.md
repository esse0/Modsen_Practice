# Meetup api

> # Before launching the application, fill out the env files

## Description

My practice project with Nest Js, postgreSQL, Prisma ORM, Swagger.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Docker deploying

```bash
# build image
$ npm run docker:build

# start/create container
$ npm run docker:start
```
